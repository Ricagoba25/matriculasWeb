# Aprendiendo Github

Este es un repositorio compartido para aprender github colaborando entre amigos.


## Abrir el proyecto en local

Clone the project

```bash
  git clone https://github.com/Ricagoba25/matriculasWeb.git
```

Go to the project directory

```bash
  cd matriculasWeb
```
